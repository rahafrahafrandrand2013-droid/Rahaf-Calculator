# Rahaf-Calculator
​"A professional scientific calculator developed in Python, featuring advanced math operations, complex number support, and robust error handling. Built by a 13-year-old student passionate about computer science." # Rahaf-Calculator

A scientific calculator developed in Python, designed to handle basic mathematical operations with error management. This project was created as part of my journey to master software development.

## Features
- Performs addition, subtraction, multiplication, and division.
- Error handling for invalid inputs (e.g., division by zero).
- User-friendly console interface.

## How to run this code
1. Ensure you have Python installed on your computer.
2. Clone this repository: `git clone https://github.com/rahafrahafrandrand2013-droid/Rahaf-Calculator.git`
3. Run the script using the command: `python calculator.py`
